# Pro Developer Portfolio

A professional, animated portfolio website built with modern HTML5, CSS3, and JavaScript (GSAP).

## Features

- **Responsive Design**: Looks great on all devices.
- **Dark Mode Aesthetic**: Professional "Engineer" style with Orange/Black palette.
- **Animations**: Powered by GSAP for smooth scrolling and interaction.
- **Interactivity**: Custom cursor, glitch text effects, and hover states.

## Customization

1.  **Edit `index.html`**: Update the text content (Name, About, Projects, etc.) to match your profile.
2.  **Images**: Place your project images in an `assets` folder (create one) and update the `src` attributes in `index.html`.
3.  **Colors**: You can tweak the colors in `style.css` under the `:root` variables.

## Hosting on GitHub Pages

1.  Create a new repository on GitHub (e.g., `my-portfolio`).
2.  Push these files to the repository.
3.  Go to **Settings** > **Pages**.
4.  Under **Source**, select `Deploy from a branch`.
5.  Select `main` (or `master`) branch and `/ (root)` folder.
6.  Click **Save**.

Your site will be live at `https://yourusername.github.io/my-portfolio/`.
